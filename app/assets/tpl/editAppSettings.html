Application settings

General

If enabled, ::app:: will automatically re-open your last project and select the last level you were in.

Visuals

This setting will affect the whole User Interface size.
This setting will only affect the size of texts and icons in the EDITOR VIEW (including entities custom fields and level custom fields).
This option controls the visual aspect of custom entity fields in the editor.
If enabled, the tiles from nearby levels will also be rendered, to help you when painting transitions between maps. Please note that this could have a significant impact on performances for large levels.
When the "single-layer" mode is enabled, this setting defines the intensity of the fading effect applied to other layers.
Set default "Full screen" state when starting ::app::. You may also toggle fullscreen mode at any time using the F11 key.
Enable this option to use color-blind friendly colors by default for newly created entities, enums or intGrid values.
If enabled, the LDtk window will be grayed out when it is not focused.

Controls

This setting affects the zoom in/out speed when using the mouse wheel.
%
This setting allows to automatically switch to the World view when zooming out from a level. You can then zoom back in to any level to select it.
Navigation keys are used to navigate through the palette of values in a layer. For example, in a Tiles layer, you can navigate through the different tiles using these keys.

Advanced options

When enabled, ::app:: will always try to use the best GPU available on your system, resulting in better performances. This might use extra battery life.
